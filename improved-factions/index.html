Extension System

This plugin ships with a economy system using Vault

Vault is no dependency, as ProtocolLib is

It is only required, when you want to use economy. To work you need to install a economy supporting plugin (like EssentailsX)

Extension System

This systems purpose is, for easy access to download addons for thie faction plugin.

I hated it, when you have a faction plugin with thousand features, but you only what some of them

This is why I wrote the Extension System. You have easy access to some features, not everybody likes

Some want to have a dynmap support and others want more settings in for faction owners

Just use /f extension download and download one of them and try them your own

All are working well together

Everything ingame

  • ChatMessageExtension -

    Adds a faction chat and shows faction next to text in the chat

  • DynmapExtension -

    Display the claimed chunks on the dynmap

  • HomeExtension -

    Adds a the /f home command

  • SettingsAndFlags -

    Add some new settings for your faction owners
