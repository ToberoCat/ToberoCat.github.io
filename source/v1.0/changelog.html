Tobero - v1.0



v1.0 - Snapshot

Updated

- Changed language file loading system. Old files will get new lines added, etc

- Backup system. If any saving process, it will put the plugin in standBy

- StandByMode. It allows the plugin to protected claimed chunks from griefing while some work is being done or the plugin crashes

- MySQL support

- Dynamic faction loading

- Config GUIs

Troubles

- DON'T use it on servers. It is NOT loading old data. It has also NO extensions, NO none-admin commands, NO permissions, etc

- There are no backups generated (Because nothing needs to be backuped). This means you can't see the backup GUI in action
